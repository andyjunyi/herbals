# 甘草的功效與搭配詳解

甘草在中醫被譽為「**國老**」，因其能調和諸藥、補脾益氣，素有「十方九草」之稱[[26]]。以下為您整理甘草的主要功效、常見搭配方式與使用注意事項：

---

## 🌿 甘草的主要功效

### 中醫傳統功效
| 功效 | 說明 |
|------|------|
| **補脾益氣** | 改善脾胃虛弱引起的倦怠乏力、食少便溏[[26]] |
| **祛痰止咳** | 稀釋痰液、促進排出，緩解咳嗽與喉嚨不適[[2]] |
| **清熱解毒** | 緩解咽喉腫痛、喉痹等熱症，改善紅腫疼痛[[1]] |
| **緩急止痛** | 緩和筋肉痙攣，適用於腹痛、胃痛等[[26]] |
| **調和藥性** | 中和其他藥材偏性，減少副作用，增強整體療效[[21]] |
| **益氣復脈** | 改善心悸、脈結代等心律不整症狀[[26]] |

### 現代藥理研究證實的功效
- 🔹 **抗發炎作用**：甘草酸能抑制炎症介質釋放，減輕組織損傷[[2]]
- 🔹 **抗病毒能力**：對肝炎病毒、疱疹病毒等具有抑制作用[[2]]
- 🔹 **免疫調節**：增強巨噬細胞與T細胞活性，提升免疫力[[2]]
- 🔹 **抗氧化保護**：捕捉自由基，保護細胞免受氧化損傷[[2]]
- 🔹 **護肝作用**：促進肝細胞再生，改善肝功能[[2]]
- 🔹 **緩解胃腸不適**：促進胃黏液分泌，保護胃黏膜[[2]]

## ⚠️ 使用禁忌與注意事項

### 不建議使用甘草的人群
1. **高血壓患者**：甘草酸可能升高血壓，加重病情[[9]][[10]]
2. **低鉀血症患者**：可能導致鉀離子流失，引發肌肉無力、心律不整[[13]]
3. **腎功能不全者**：鈉水瀦留可能加重腎臟負擔[[9]]
4. **心血管疾病患者**：可能影響心律與血壓穩定[[9]]
5. **孕婦與哺乳期婦女**：可能影響激素水平，建議諮詢醫師後使用[[9]][[26]]
6. **術前兩週內**：可能增加出血風險，建議暫停使用[[26]]
7. **正在服用特定藥物者**：如利尿劑、強心苷、降壓藥、類固醇等，可能產生交互作用[[12]][[26]]

### 用量與使用建議
- 📏 **每日建議量**：乾品甘草根2～4克，甘草酸不超過100毫克[[2]]
- ⏱️ **避免長期大量使用**：可能引起水鈉瀦留、低血鉀、水腫等副作用[[14]]
- 🚫 **配伍禁忌**：甘草反大戟、芫花、甘遂、海藻，不宜同用[[13]][[26]]

## 💡 小提醒

> 甘草雖為「國老」，但使用仍需**辨證論治**。建議在中醫師或專業醫療人員指導下搭配使用，尤其是長期保健或合併用藥時，務必諮詢專業意見，以確保安全與療效[[2]][[26]]。
