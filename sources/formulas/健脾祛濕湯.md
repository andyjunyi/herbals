# 健脾祛濕湯（代茶飲）

**來源：玉林市中醫院**  
**發布者：@藥評君李不同**（第60集／真實無廣測評）

---

## 🌿 藥方與劑量（黃色字體辨識結果）

| 藥材 | 劑量 |
|------|------|
| 山藥 | 6 克 |
| 蓮子 | 6 克 |
| 炒白扁豆 | 6 克 |
| 赤小豆 | 6 克 |
| 茯苓 | 6 克 |
| 陳皮 | 3 克 |

> ✅ 總藥材量約 33 克，建議加水 500–800ml。

---

## 🍵 調製方法（二選一）

### 🔹 簡易版（保溫杯／悶泡）
1. 藥材快速沖洗乾淨  
2. 放入保溫杯，加入 **沸水 500–800ml**  
3. 蓋緊悶泡 **30 分鐘** 後即可飲用  
4. 可反覆加水至無味，當日喝完  

### 🔹 效果更佳版（養生壺／煎煮）
1. 藥材洗淨，加水 **800ml** 浸泡 15–20 分鐘  
2. 大火煮滾，轉小火再煮 **15–20 分鐘**  
3. 濾渣取湯，溫熱飲用  
4. 可再加水回煮 1 次  

> 💡 可加少許紅棗或冰糖調味，但不加效果更純。

---

## 💪 主要功效

### 核心作用：**健脾 + 祛濕 + 理氣**

| 藥材 | 功效 |
|------|------|
| 山藥 | 補脾養胃 |
| 蓮子 | 補脾止瀉、安神 |
| 炒白扁豆 | 健脾化濕 |
| 赤小豆 | 利水消腫、清熱祛濕 |
| 茯苓 | 利水滲濕、健脾 |
| 陳皮 | 理氣健脾、燥濕化痰 |

### 👥 最適合這些人
- 容易疲勞、身體沉重  
- 大便黏馬桶、不成形  
- 舌苔厚膩、口黏  
- 飯後腹脹、食慾不佳  
- 頭臉易出油、輕微水腫  

---

## ⚠️ 注意事項與禁忌

### ✅ 適合飲用時機
- 梅雨季、夏季濕氣重  
- 久坐少動、常吃生冷油膩  
- 感覺濕氣重、身體沉重時  

### ❌ 不適合或需諮詢醫師
- 陰虛火旺（口乾舌燥、手腳心熱、舌紅少苔）  
- 嚴重便祕、大便乾結  
- 急性發燒、喉嚨痛、黃痰  
- 懷孕初期（請諮詢中醫師）  

### 🚫 飲用期間飲食建議
- 少吃：冰品、甜食、油炸、西瓜、芒果  
- 適宜：溫熱飲食（薑茶、熟食）  

---

## 📅 建議飲用頻率
- 連續 **5～7 天**，或每週 **3～4 次**  
- 不建議隔夜飲用
