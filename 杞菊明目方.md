# 杞菊明目方（代茶飲）

**來源**：雲南省中醫院  
**發布者**：@藥評君李不同（第51集）  
**標題**：9毛8一袋？真實無廣測評

---

## 🌿 藥方與劑量

| 藥材 | 劑量 |
|------|------|
| 谷精草 | 2 克 |
| 山萸肉 | 1 克 |
| 菊花 | 2 克 |
| 枸杞 | 3 克 |
| 甘草 | 1 克 |

> ✅ 總藥材量約 **9 克**，建議加水 **300–400ml**

**備註**：圖片上半部收費明細中另有「決明子、陳皮、炒白朮、砂仁、茯苓、黨參、黃芪、山藥、金銀花、蒲公英」等藥材，但**不屬於此配方**，可能是該集測評中的對比或額外資訊。

---

## 🍵 調製方法

### 簡易版（保溫杯／悶泡）
1. 藥材快速沖洗
2. 放入保溫杯，加入 **沸水 300–400ml**
3. 悶泡 **10–15 分鐘** 後飲用
4. 可回沖 1–2 次

### 效果更佳版（養生壺／煎煮）
1. 藥材洗淨，加水 **400ml** 浸泡 5–10 分鐘
2. 大火煮滾，轉小火再煮 **5–8 分鐘**
3. 濾渣取湯，溫熱飲用

> 💡 甘草微甜，此方無需額外加糖。

---

## 💪 主要功效

| 藥材 | 功效 |
|------|------|
| 谷精草 | 疏散風熱、明目退翳 |
| 山萸肉 | 補益肝腎、收斂固澀 |
| 菊花 | 清肝明目、疏散風熱 |
| 枸杞 | 滋補肝腎、益精明目 |
| 甘草 | 調和諸藥、益氣和中 |

### 核心作用：**明目 + 補肝腎 + 清熱**

### 👥 適合對象
- 用眼過度（電腦、手機族）
- 眼睛乾澀、疲勞、視物模糊
- 肝腎不足引起的頭暈眼花
- 輕微風熱目赤（眼睛紅、畏光）

---

## ⚠️ 注意事項與禁忌

### ✅ 適合飲用時機
- 長時間用眼後
- 春季養肝明目
- 眼睛疲勞乾澀時

### ❌ 不適合或需諮詢醫師
- 脾胃虛寒、容易腹瀉者（山萸肉偏收斂，谷精草微寒）
- 感冒發燒、喉嚨痛急性期
- 孕婦（建議諮詢中醫師）
- 對菊花、谷精草過敏者

### 🚫 飲用期間飲食建議
- 少吃：辛辣油炸、燒烤、過甜食物
- 適宜：胡蘿蔔、藍莓、綠色蔬菜

---

## 📅 建議飲用頻率
- 每週 **3～5 次**，連續 2 週後可休息 3–5 天
- 不建議隔夜飲用
