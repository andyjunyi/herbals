# Claude 藥方整理原則

## 核心流程
1. 辨識圖片中所有文字（尤其特殊顏色如黃色字體）
2. 提取：藥名 + 劑量 → 製成表格
3. 補充：調製方法、功效、禁忌

## 輸出格式（固定順序）
- 藥方表格（藥材｜劑量）
- 調製方法（簡易版悶泡 ＋ 效果更佳版煎煮）
- 功效表格（藥材｜功效）＋ 適合對象條列
- 注意事項（適合時機、禁忌、飲食建議）
- 飲用頻率 ＋ 免責聲明

## 藥方表格規則
| 藥材 | 劑量 |
|------|------|
| 名稱 | 克數 |

- 總藥材量加總，建議水量 500–800ml
- 劑量未標示 → 註明「未標示」

## 調製方法簡寫範本
**簡易版**：洗 → 保溫杯＋沸水500–800ml → 悶30分鐘  
**煎煮版**：洗 → 加水800ml浸泡15分鐘 → 煮滾轉小火15–20分鐘 → 濾渣

## 功效表格範例
| 藥材 | 功效 |
|------|------|
| 山藥 | 補脾養胃 |

核心作用：2–4 個關鍵詞粗體（如 **健脾、祛濕**）

## 禁忌固定項目
- 陰虛火旺、嚴重便祕、急性發燒、懷孕初期（諮詢醫師）
- 少吃：冰品、甜食、油炸、西瓜、芒果
